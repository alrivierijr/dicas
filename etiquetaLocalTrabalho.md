# Mínimo de Etiqueta no Local de Trabalho
## Evitar :

1. Cantar, batucar na mesa com o dedo/unha / caneta ou bater os pés acompanhando a música ambiente
```
Vá ser cantor ou baterista, se este é o seu caso.
```
2. Não passar desodorante / creme de mão / perfume após o almoço
```
Ninguém é obrigado a aguentar o cheiro forte de perfume durante a digestão
```
3. Não falar alto no telefone, tirando o foco dos que estão próximo
```
Não atrapalhe a concentração de quem está querendo trabalhar
```
3. Quando ligar para alguém, informe seu nome/depto. assim que atenderem.
```
Telefone não é video-conferência
```
4. Quando cumprimentar, olhe no olho, não seja mais falso do que já é
5. Quando apertar a mão para cumprimentar, aperte forte, ou dê a mão para o outro beijar
6. Ao espirrar ou tossir, use lenço para barrar a baba e/ou o ranho, ninguém é obrigado a respirar isso.
